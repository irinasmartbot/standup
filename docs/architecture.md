# Moscow StandUp Show — Архитектура проекта

## Общая схема системы

```
Google Таблица (Афиша)
        ↓
    Telegram Бот  ←→  База данных (SQLite → PostgreSQL)
        ↓
    Пользователь
```

**Текущее состояние:** бот читает мероприятия из Google Таблицы напрямую при каждом запросе.  
**Следующий шаг:** мероприятия переедут в базу данных, синхронизация с таблицей — автоматическая.  
**В будущем:** Google Таблица заменяется на Админ-панель для менеджера.

## Структура проекта

```
standup/
├── main.py                      # Точка запуска бота
├── bot/
│   ├── config.py                # Настройки и переменные окружения
│   ├── db/
│   │   ├── models.py            # Структура таблиц базы данных
│   │   └── crud.py              # Операции с базой данных
│   ├── services/
│   │   └── sheets.py            # Загрузка данных из Google Таблицы
│   ├── utils/
│   │   └── ticket.py            # Генерация билета, вспомогательные функции
│   └── handlers/
│       ├── start.py             # Команда /start, главное меню
│       ├── formats.py           # Форматы шоу, площадки, правила
│       ├── booking.py           # Воронка бронирования
│       └── reminders.py         # Напоминания и аннулирование броней
├── docs/                        # Документация проекта
├── requirements.txt             # Зависимости Python
├── .env.example                 # Пример файла настроек
└── .gitignore
```

## Технологический стек

| Компонент | Технология |
|---|---|
| Язык | Python 3.11+ |
| Telegram | aiogram 3 |
| База данных (сейчас) | SQLite |

## Переменные окружения (.env)

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен Telegram-бота (от @BotFather) |
| `CSV_URL` | Публичная ссылка на CSV из Google Таблицы |
| `MANAGER_LINK` | Ссылка на менеджера в Telegram |
| `CHANNEL_LINK` | Ссылка на канал анонсов |
| `MANAGER_PHONE` | Телефон менеджера |
| `DB_PATH` | Путь к файлу базы данных |
| `TICKET_TEMPLATE` | Путь к шаблону билета (изображение) |

## Форматы шоу

### StandUp Проверка материала
- Вход бесплатный
- 5–7 комиков по 10–15 минут
- Воронка бронирования реализована полностью
- Подробнее: [flows/proverka_materiala.md](flows/proverka_materiala.md)

### StandUp BEST
- Билеты от 500 рублей
- Воронка бронирования — в разработке
- Временно: направление к менеджеру

### Форматы в плане
- 1+1
- Мастер-классы (с оплатой и проверкой чека)
- Хит-лото
- Розыгрыши из канала
- Сертификаты (актуальность под вопросом)
