# Moscow StandUp Show Telegram Bot

Telegram-бот для бронирования мест на мероприятия Moscow StandUp Show.

## Что уже есть

- Главное меню с разделами: бронь, форматы, площадки, правила, менеджер, канал.
- Ветка `StandUp Проверка материала`.
- Загрузка актуальной афиши из опубликованной Google-таблицы CSV.
- Выбор мероприятия по дате или площадке.
- Бронирование: имя, телефон, количество гостей.
- Ограничение до 4 гостей на одну бронь.
- Проверка лимита мест по мероприятию.
- Отмена брони, смена даты, изменение количества гостей.
- Генерация билета через Pillow.
- Напоминания и аннулирование неподтвержденных броней.

## Чего еще нет

- Полная ветка `StandUp BEST`.
- Ветка формата `1+1`.
- Ветка мастер-классов с оплатой и проверкой чека.
- Ветка розыгрыша из Telegram-канала.
- Админ-панель или выгрузка броней для менеджера.

## Запуск

1. Установить Python 3.11+.
2. Создать виртуальное окружение и поставить зависимости:

```bash
pip install -r requirements.txt
```

Если команда `python` не находится, используйте установленный Python напрямую:

```powershell
& "$env:LOCALAPPDATA\Programs\Python\Python314\python.exe" -m pip install -r requirements.txt
```

3. Скопировать `.env.example` в `.env` и заполнить `BOT_TOKEN`.
4. Запустить:

```bash
python bot.py
```

Или через полный путь:

```powershell
& "$env:LOCALAPPDATA\Programs\Python\Python314\python.exe" bot.py
```

База SQLite создается автоматически в `bookings.db`.

## Админ-панель броней

Визуальная админка показывает мероприятия, заполненность, статусы броней и список клиентов.

```bash
python admin.py
```

По умолчанию страница доступна на `http://127.0.0.1:8080/admin`.
Для сервера задайте в `.env` `ADMIN_HOST`, `ADMIN_PORT` и `ADMIN_TOKEN`.
Если `ADMIN_TOKEN` заполнен, при открытии `/admin` появится форма входа.
Также можно один раз открыть `/admin?token=ваш_токен`: админка сохранит защищенную cookie и уберет токен из адресной строки.
`ADMIN_TOKEN` — для менеджеров (брони и клиенты).
`ADMIN_OWNER_TOKEN` — для владельца: дополнительно вкладка `База` с просмотром таблиц.

## Важное

Не храните реальный Telegram-токен в `bot.py` и не отправляйте `.env` в публичные места.
